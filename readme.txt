== Description ==

WordPress plugin for image optimization, which uses the service Google PageSpeed Insights.

When you check the speed of a site on the Google Page Speed,
Google often gives out which images need to be optimized.

To do this, we created a plugin Just Image Optimizer that communicates with the service and receives optimized images and replaces them on your site.
You can do this in manual and automatic mode, you only need to configure the plugin.

With this plugin you get good results in image optimization on Google Page Speed.

This is a good solution for optimizing images on your website.

== Installation ==

1. Download, unzip and upload to your WordPress plugins directory
2. Activate the plugin within you WordPress Administration Backend
3. Configure the plugin

== Upgrade Notice ==

To upgrade remove the old plugin folder. After than follow the installation steps 1-3.
